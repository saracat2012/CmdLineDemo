# Git Basics
* Git started with Git
```
git init -- initialize git tracking
```

* What's going on?
```

git status -- get current status of repo
```

* Stage 1 file or all files
``` 
git add readme.md
git add -A
```

* Commit
```
git commit -m "Initial commit"
```

* Add origin
```
git remote add origin 'url'
```

* Push to github
```
git push origin master
```
